# Quick Test Guide

## Installation
```bash
npm install
```

## Running Tests

```bash
npm test              # Run all tests once
npm run test:watch    # Watch mode (auto-rerun on changes)
npm run test:ui       # Interactive UI mode
npm run test:coverage # Generate coverage report
```
